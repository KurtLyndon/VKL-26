# Developer Docs

Thư mục này dành cho tài liệu kỹ thuật phục vụ phát triển.

## Nên đọc khi nào

- viết hoặc sửa script cho agent
- sửa migration/schema
- triển khai contract runtime agent
- rà lịch sử thay đổi kỹ thuật của dự án

## Danh sách tài liệu

- `WORKLOG.md`: nhật ký triển khai kỹ thuật theo thời gian
- `MIGRATIONS.md`: quy ước migration và cách chạy
- `AGENT_RUNTIME_CONTRACT.md`: contract runtime giữa backend và external agent
- `POC_SCRIPT_CONTRACT.md`: chuẩn cho script PoC xác minh finding/CVE
- `TASK_SCRIPT_CONTRACT.md`: chuẩn cho script task do agent thực thi
