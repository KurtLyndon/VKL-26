# Cath Docs

Thư mục này dành riêng cho Cath/Codex để dùng làm ngữ cảnh nhanh cho các phiên tiếp theo.

## Quy tắc sử dụng

- Mặc định chỉ đọc tài liệu trong `docs/cath` trước khi bắt đầu làm việc.
- Chỉ mở thêm tài liệu trong `docs/developer` khi tác vụ thực sự cần chi tiết kỹ thuật.
- Không dùng `docs/developer/WORKLOG.md` làm tài liệu đọc mặc định nếu chưa cần.

## Thứ tự đọc khuyến nghị

1. `NEXT_SESSION.md`
2. Nếu cần bối cảnh kỹ thuật sâu hơn, mới mở:
   - `../developer/WORKLOG.md`
   - `../developer/AGENT_RUNTIME_CONTRACT.md`
   - `../developer/POC_SCRIPT_CONTRACT.md`
