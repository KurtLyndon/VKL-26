# User Docs

Thư mục này dành cho tài liệu phục vụ người dùng và người vận hành hệ thống.

Hiện tại phần lớn hướng dẫn khởi động local vẫn đang nằm ở:

- `README.md` tại thư mục gốc project

Các tài liệu user-facing chi tiết hơn trong tương lai nên được đặt tại đây, ví dụ:

- hướng dẫn import scan
- hướng dẫn thao tác dashboard
- hướng dẫn quản lý target, finding, report
